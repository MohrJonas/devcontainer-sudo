# Devcontainer-sudo

This repository contains a feature for devcontainer called "devcontainer-sudo".  

## About

Devcontainer-sudo installs sudo and allows passwordless sudo access for the specified user.

## Example

```json
"features": {
    "ghcr.io/MohrJonas/devcontainer-sudo/devcontainer-sudo:1": {
        "user": "containerUser"
    }
}
```

## Compatibility

Devcontainer-sudo is designed to work on distributions using `apt`, `dnf`, `apk` and `pacman`.
